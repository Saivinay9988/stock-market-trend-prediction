# Stock Market Trend Prediction Using Sentiment Analysis

This project is a web-based application that predicts stock market trends (UP/DOWN) by analyzing the sentiment of financial news text.

## Technologies Used
- Python
- Flask
- TextBlob (NLP)
- HTML
- Git & GitHub

## How It Works
1. User enters stock-related news.
2. Sentiment analysis is performed using NLP.
3. Based on sentiment polarity, trend is predicted as UP or DOWN.

## How to Run the Project
1. Clone the repository
2. Create virtual environment
3. Install dependencies
4. Run the Flask app

```bash
pip install -r requirements.txt
python app.py
